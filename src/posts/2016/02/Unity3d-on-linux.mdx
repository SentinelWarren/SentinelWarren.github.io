Baada ya kucheza na Serious Bugs za experimental builds zilizopita,finally hii latest build imeanza kuleta muangaza wa uwezekano wa hii fantastic Project kufanikiwa.
Binafsi nimeanza kutumia hii engine pale walipo release first public build{2015082501},haikua mbaya ila pia ilikua very unstable!,ukitumia within 25 minutes haijacrash,lazima ushangilie,so far ilikua inatufaa sie hobbyist Game Devs tu,ila kwa production purpose,uwezi fanya kitu cha maana,kuload izo standard asset tu ilikua shughuli!ila so far baada ya some builds kupita,hizi two latest build zimeanza kuonesha matumaini mapya ya project kuwezekana because,huko nyuma walishawahi jaribu ikashindikana.
So far,any 2D production unaweza fanya kwa linux bila hata haja ya ku dual boot,ila still bado kwa heavy 3D production hakuna ujanja zaidi ya kufanyia kazi kwa windoze au mac!

Kwenye hii latest build , 
- WEBGL problems ziimekwisha kabisa(latest build nilipokua nata kubuild webgl app unity ilikua inacrash)
- MonoDevelop inalaunchika from the editor bila tatizo,past build unity ilikua lazima i-crash
- Gray webview hakuna tena on first launch
- Ilikua haisomi Blender extension(blend) kwenye kuimport.Pia Ipo very smooth now.

Ila so far Unreal Engine iko stable zaidi kwa linux machine ukilinganisha na Unity3D.

Release Notes

- Fix VR publishing for supported platforms (e.g. GearVR)
- Fix menu item invocation after play/stop cycle
